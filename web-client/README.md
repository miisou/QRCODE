# Web Client

Frontend for GovVerify MVP.

## Setup

1.  Install dependencies: `npm install`

## Running the App

Start development server:
```bash
npm run dev
```

## Running Tests

Currently, the web client does not have a dedicated test suite.
Linting can be run via:
```bash
npm run lint
```
App runs on `http://localhost:5173`.

## Stopping

Press `q` or `Ctrl+C` in the terminal.
